Assumptions & basis of computation

  • Source figures — the {{ period_label }} audited balance sheet ({{ auditor }}). The limit should be re-cast on each fresh audited statement.
  • Existing bank finance — {% if has_existing_bank %}{{ existing_bank_inr }} netted from current liabilities.{% else %}none on record, so other current liabilities are taken as the full current-liabilities balance. An existing CC/OD, once recorded, would net against this.{% endif %}
  • Drawing-power margins — {{ stock_margin_pct }} on paid stock and {{ debtor_margin_pct }} on book debts, per standard CC norms. Sundry creditors are deducted from stock to value only paid-for inventory.
  • Debtor eligibility — ageing (the usual ≤90-day cut-off) is not on file, so the full debtor balance is taken net of margin. Tightening on overdue debtors would reduce drawing power, not the MPBF-based limit.
